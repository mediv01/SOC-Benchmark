# SOC-Benchmark
SOC-Benchmark   
手机处理器天梯图

本图收录了近几年常见手机SOC，包括高通骁龙、海思麒麟、苹果、三星和联发科。
本图参考 GeekBench 的跑分数据，以骁龙425的单核和多核性能作为100%基准
